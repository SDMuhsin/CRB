# Tuned for Alliance Canada wheelhouse compatibility (Nibi, Python 3.11).
#
# CRITICAL — Qwen3 support requires transformers >= 4.51.0 (April 2025). If
# the Alliance wheelhouse does not yet ship a transformers >= 4.51, the
# `transformers` / `tokenizers` / `huggingface_hub` trio MUST be installed
# from PyPI (i.e. drop `--no-index` for those). Previous pin of >=4.42
# silently resolved to a wheel that lacks Qwen3ForCausalLM and produced:
#     ModuleNotFoundError: Could not import module 'Qwen3ForCausalLM'.
#
# Install on a Nibi compute env:
#     module load gcc arrow scipy-stack cuda cudnn python/3.11
#     virtualenv --no-download env && source env/bin/activate
#     pip install --no-index --upgrade pip
#     # Try wheelhouse first:
#     pip install --no-index -r requirements.txt || true
#     # Then force PyPI for Qwen3-capable transformers (wheelhouse may lag):
#     pip install --upgrade 'transformers>=4.51' 'tokenizers>=0.21' 'huggingface_hub>=0.25'
#     # flash1dkmeans is not in the wheelhouse either:
#     pip install flash1dkmeans
#
# Verify Qwen3 is importable before submitting sbatch jobs:
#     python -c "from transformers.models.qwen3 import Qwen3ForCausalLM; print('ok')"
